EXP-8 This project implements a simple REST API using Spring Boot that allows management of student data through basic CRUD operations. It uses Spring Data JPA to interact with a MySQL database, enabling efficient storage and retrieval of records. The API exposes endpoints for creating, reading, updating, and deleting student information using standard HTTP methods. The application is developed in Java 17, built with Maven for dependency management, and implemented in Eclipse IDE, ensuring a structured and scalable backend service.
