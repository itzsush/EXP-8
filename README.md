EXP-8 This project implements a simple REST API using Spring Boot that allows management of student data through basic CRUD operations. The API exposes endpoints for creating, reading, updating, and deleting student information using standard HTTP methods. The application is developed in Java 17 and built with Maven, and it runs using an embedded server without connecting to a database.
